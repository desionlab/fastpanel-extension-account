# fastPanel Account.
Add-on to work with users of the system.

---

## Events

### account:login
```typescript
  this.events.on('account:login', async (user: IUser) => {});
```

### account:logout
```typescript
  this.events.on('account:logout', async (user: IUser) => {});
```
